# 🔔 School Bell System

A professional, user-friendly automated bell management system specially designed for schools. Built using Python and Tkinter, this application provides a modern GUI to manage school schedules efficiently.

![Python](https://img.shields.io/badge/python-3.x-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

## ✨ Key Features

* **User-Friendly GUI:** Simple and intuitive interface for non-technical users.
* **Customizable Timetables:** Create, edit, and manage multiple bell schedules easily.
* **Audio Support:** Play high-quality MP3 or WAV files for school bells.
* **Background Operation:** Runs in the system tray to save taskbar space.
* **High Priority Execution:** Optimized to ensure the bell rings precisely on time.
* **Auto-Backup:** Automatically backs up timetable settings to prevent data loss.

## 🚀 How to Use

### For General Users
1.  Go to the **[Releases](https://github.com/YOUR_GITHUB_USERNAME/school-bell-system/releases)** section.
2.  Download the latest `School_Bell_Setup.exe`.
3.  Install the application and launch it from the desktop shortcut.

### For Developers (Run from Source)
1.  Clone this repository:
    ```bash
    git clone [https://github.com/YOUR_GITHUB_USERNAME/school-bell-system.git](https://github.com/YOUR_GITHUB_USERNAME/school-bell-system.git)
    ```
2.  Install required libraries:
    ```bash
    pip install psutil pillow pygame pystray
    ```
3.  Run the application:
    ```bash
    python main.py
    ```
