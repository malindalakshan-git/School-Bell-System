# 🔔 School Bell System

A professional, user-friendly automated bell management system specially designed for schools. Built using **Python** and **Tkinter**, this application provides a modern GUI to manage school schedules efficiently.

![Python](https://img.shields.io/badge/Python-3.x-blue.svg)
![License](https://img.shields.io/badge/License-MIT-green.svg)

---

## ✨ Key Features

* **User-Friendly GUI:** Simple and intuitive interface for non-technical users.
* **Customizable Timetables:** Create, edit, and manage multiple bell schedules easily.
* **Audio Support:** Play high-quality MP3 or WAV files for school bells.
* **Background Operation:** Runs smoothly in the background to ensure no bell is missed.
* **High Priority Execution:** Optimized to ensure the bell rings precisely on time.
* **Auto-Save:** Automatically saves your timetable settings to prevent data loss.

---

## 🚀 How to Use

### For General Users (Quick Start)
1.  Go to the **[Releases](https://github.com/malindalakshan-git/School-Bell-System/releases)** section.
2.  Download the latest `School.Bell.System_setup.exe`.
3.  Install the application and launch it from the desktop shortcut.
4.  Go to **Settings** -> **Create New Timetable** to start.

### For Developers (Run from Source)
1.  **Clone this repository:**
    ```bash
    git clone [https://github.com/malindalakshan-git/School-Bell-System.git](https://github.com/malindalakshan-git/School-Bell-System.git)
    ```
2.  **Install required libraries:**
    ```bash
    pip install psutil pillow pygame pystray
    ```
3.  **Run the application:**
    ```bash
    python main.py
    ```

---

## 🛠 Configuration Guide

* **Create:** Click "Create New Timetable", give it a name, add periods with times, and select your bell sounds.
* **Save:** Don't forget to click the **Save** button.
* **Activate:** Go to "Open Saved Timetable", select your file, and click **Start** to run the system.
